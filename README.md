# Food Sensitivity App - GitHub Ready

هذا المجلد جاهز للرفع مباشرة إلى GitHub Pages.

## الملفات المهمة
- `index.html` الواجهة الرئيسية
- `manifest.webmanifest` إعدادات PWA
- `sw.js` Service Worker
- `config.js` إعدادات اختيارية للربط مع API خارجي لاحقاً
- `icon-192.png` و `icon-512.png` أيقونات التطبيق

## طريقة النشر على GitHub Pages
1. أنشئ Repository جديد على GitHub.
2. ارفع جميع الملفات الموجودة في هذا المجلد إلى جذر الريبو.
3. من إعدادات الريبو افتح Pages.
4. اختر Branch الرئيسي ثم `/root`.
5. احفظ الإعدادات وانتظر إنشاء الرابط.

## ملاحظة مهمة
هذه النسخة تعمل بالكامل كواجهة ثابتة على GitHub Pages.
إذا أردت لاحقاً بحثاً حياً عبر الإنترنت عن المكونات أو تحليل منتجات لحظي، ستحتاج Backend / API خارجي ثم تضع الرابط في `config.js`.
